# Inflect Compliance

Inflect Compliance is a multi-tenant **Governance, Risk & Compliance (GRC)** platform for managing controls, risks, evidence, audits, and vendors across multiple frameworks.

Supported frameworks include ISO 27001, ISO 42001, SOC 2, NIST CSF, NIST
Privacy Framework, NIST SSDF, NIS2, DORA, and the EU AI Act.
