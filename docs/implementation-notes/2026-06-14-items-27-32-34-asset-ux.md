# 2026-06-14 — items 27 / 32 / 34: asset list interaction overhaul

**Branch:** `claude/feat-27-32-34-asset-ux`

Three asset-list changes that share the `AssetsClient` surface, shipped
together.

## Item 34 — Criticality column

The asset table gained a **Criticality** column derived live from the
row's C/I/A via `getAssetCriticality` (the item-25 top-two-mean model),
rendered as a toned `StatusBadge`. No API change — the list already
returns C/I/A. Column-set decision: criticality leads the
risk-relevant columns and `classification` is demoted to off-by-default
(criticality is the operative prioritisation signal and largely
subsumes classification). The gear still exposes both.

## Item 32 — Single-click quick-look panel

A single row click now opens a right-side `<Sheet>` quick-look
(`AssetDetailSheet`) instead of navigating to the full page — the same
inspect-without-losing-context pattern evidence + controls already use.
The panel renders from the row object already in hand (no fetch) and
carries a **Full view →** button (footer, bottom-right) that enters
`/assets/[id]` when the user wants the tabs + edit surface. The name
cell is now a plain (non-link) title so the whole row speaks one
interaction; the old `asset-link-<id>` navigation Link is gone.

## Item 27 — Arrow-key navigation

While the panel is open, ↑/↓ move the selection to the previous/next
asset in the current filtered+sorted list (clamped at the ends), so a
reviewer can walk the register without the mouse. Registered via
`useKeyboardShortcut('ArrowUp' | 'ArrowDown', …, { enabled: panelOpen })`
— disabled when the panel is closed so it never competes with table
scroll.

| File | Role |

## Decisions

- **Pass the row, don't refetch.** The list row carries everything the
  panel shows, so the panel is instant and the detail GET is reserved
  for the full-view page.
- **Whole-row interaction.** Dropping the name Link (vs. keeping it as a
  competing full-page nav) means click-anywhere → panel is consistent;
  full-page is one deliberate button away.
- **Arrow nav gated on panel-open.** Avoids hijacking ArrowUp/Down when
  the user is just scrolling the table.
